# 🐱 Desmos Cat

A mathematical cat drawn entirely using equations on [Desmos](https://www.desmos.com/calculator/7zlg4tahzx).

---

## ✨ Overview
This project demonstrates how algebraic, trigonometric, and piecewise equations can be combined to create art.  
The result is a playful cat graph, showcasing the creative side of mathematics.

---

## 🔗 Live Graph
Explore the interactive graph here:  
👉 [Desmos Cat](https://www.desmos.com/calculator/7zlg4tahzx)

---

## 🛠️ How It Works
- **Equations as shapes:** Circles, ellipses, parabolas, and lines form the cat’s body.
- **Piecewise functions:** Restrict curves to specific regions for ears, whiskers, and paws.
- **Symmetry:** Reflections across the y-axis create balanced features.
- **Layering:** Equations are ordered to overlap correctly, giving depth to the drawing.

---

## 📂 Structure
- **Body equations** – torso and head  
- **Facial features** – eyes, nose, whiskers  
- **Accessories** – tail, optional background  

---

## 🚀 Getting Started
1. Open the [Desmos link](https://www.desmos.com/calculator/7zlg4tahzx).
2. Toggle equations on/off to see how each part contributes.
3. Experiment by modifying coefficients to reshape the cat.
